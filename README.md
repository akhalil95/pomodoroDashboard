# Pomodoro Dashboard

This is a pomodoro dashboard. The backgound chages everytime you open it. You can aslo keep track of your list items.


## Features

- Unsplash API
- Current time API
- Local weather API in celsius 
- To-do list
- Pomodoro that lets you change color and font
